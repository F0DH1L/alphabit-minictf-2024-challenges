# Alphabit MiniCTF 2024

In the 2023/2024 season, I was a member of the technical team at Alphabit Club, a scientific club at my university. I helped the team build a CTF for our members.

This repo has the challenges, write-ups, and deployment scripts for the CTF.

The CTF had many categories: Web Security, Cryptography, Forensics, Misc, Pwn, Reverse Engineering.

A big thanks to all the authors who contributed.
